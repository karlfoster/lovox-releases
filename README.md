# Lovox Releases

Update feed and release downloads for [Lovox](https://github.com/karlfoster/Lovox) — a macOS push-to-talk voice transcription app.

## How it works

This repo hosts:
- **`appcast.xml`** — Sparkle update feed (served via GitHub Pages)
- **Release assets** — DMG files attached to GitHub Releases

The Lovox app checks `appcast.xml` for updates using [Sparkle](https://sparkle-project.org/).

## Feed URL

```
https://karlfoster.github.io/lovox-releases/appcast.xml
```
